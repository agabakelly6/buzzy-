## ✨ What This Site Does

Kelly Accessories is a fully responsive, multi-page business website built with pure HTML & CSS. It lets customers browse products, check prices, and reach out instantly via WhatsApp or Instagram — no app download needed.

## 🎨 Features

- 🌗 **Dark / Light Mode Toggle** — saves preference across sessions
- ✨ **Scroll Reveal Animations** — cards fade in as you scroll
- ⚡ **Page Loader** — smooth branded intro on every page load
- 📱 **Mobile Bottom Navigation** — app-like nav bar on small screens
- 💬 **WhatsApp Integration** — every product links to a pre-filled WhatsApp order message
- 📸 **Instagram Link** — direct link to [@root.kelly.404](https://instagram.com/root.kelly.404)
- 🌟 **Customer Reviews** — per-page testimonials from real customers
- 🔗 **Related Products** — cross-page suggestions to keep customers browsing
- 🎯 **About Section** — trust-building stats and store story

## 🚀 How to Deploy on GitHub Pages

1. Push all files to your GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
5. Your site goes live at `https://agabakelly6.github.io/buzzy-`
